# OKSLOP Claude Plugins

Official plugin marketplace for [OKSLOP](https://okslop.com) — stock images for Claude Code.

## Install

```
/plugin marketplace add OKSLOP/claude-plugins
/plugin install okslop@OKSLOP-claude-plugins
```

Run these inside Claude Code (not your terminal).

## What you can do

After installing, just ask Claude for what you need:

- "Add a hero image to my landing page"
- "Find me a photo of a cozy office"
- "Generate a flat-lay image of desk accessories"
- "I need a cover photo for this blog post"
- "Replace the placeholder with a real photo"

Claude searches millions of free AI-generated images, picks the best match, and inserts the right code. No commands to memorize, no image editing.

Need something that doesn't exist? Claude can generate custom images from your description.

## Available Plugins

| Plugin | Description |
|--------|-------------|
| `okslop` | Search and generate AI stock images — just by asking |
